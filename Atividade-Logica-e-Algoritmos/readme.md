# Projeto – Pensamento Computacional: Controle de Acesso

## Descrição
Este projeto foi desenvolvido como parte da atividade prática da disciplina Pensamento Computacional (UDF), com a Profa. Kadidja Valéria.

O objetivo é aplicar os pilares do pensamento computacional (decomposição, abstração, reconhecimento de padrões e algoritmos) na criação de um sistema lógico de controle de entrada.

## Objetivos
- Relacionar a lógica de programação com situações cotidianas.
- Aplicar estruturas lógicas de controle (Se-Então-Senão) e repetição.
- Desenvolver uma solução clara e sistemática para gestão de fluxo de pessoas.

## Sistema Proposto
**Nome do Sistema:** Verificador de Acesso Acadêmico  
**Descrição:** Uma solução lógica para organizar a entrada de alunos em sala de aula, garantindo segurança e organização através da checagem em listas oficiais.

## Pensamento Computacional Aplicado
- **Decomposição:** - Recebimento do nome.
  - Verificação na base de dados (lista).
  - Autorização ou negação do acesso.
- **Reconhecimento de Padrões:** - Identificação de que o processo de verificação é idêntico para todos os alunos na fila.
- **Abstração:** - Foco apenas no nome para validação, desconsiderando outras características não relevantes para o acesso.
- **Algoritmos:** - Implementação de um fluxo contínuo de verificação usando laços de repetição.

## Metodologia de Desenvolvimento
- **Metodologia:** Pensamento Computacional (PBL).
- **Ferramentas:** GitHub para versionamento e documentação em Markdown.

## Desafios Identificados
- Garantir que nenhum aluno da fila seja ignorado (resolvido com o laço de repetição).
- Tratar nomes que não constam na lista (resolvido com a condicional negativa).
