# Projeto – Prática de Lógica e Algoritmos

## Descrição
Este projeto apresenta o desenvolvimento de um algoritmo de controle de acesso, focado na aplicação prática de estruturas condicionais e variáveis.

## Objetivo
* Desenvolver a capacidade de criar pseudocódigos estruturados.
* Aplicar testes de mesa para validação de lógica de programação.

## Conteúdo
* **Código:** Algoritmo de verificação de login e senha.
* **Teste de Mesa:** Demonstração do comportamento das variáveis passo a passo.
