# Documentação: Sistema de Controle de Acesso

## Descrição
Este projeto aplica os pilares do pensamento computacional para resolver o desafio de organizar a entrada de alunos em uma sala de aula.

## Pensamento Computacional Aplicado
- **Decomposição:** Divisão entre coletar o nome, verificar na lista e decidir o acesso.
- **Reconhecimento de Padrões:** Uso de repetição para processar uma fila inteira de pessoas.
- **Abstração:** Foco apenas no dado "Nome" para validação, ignorando outros detalhes.
- **Algoritmo:** Sequência de passos estruturados para automação do processo.
