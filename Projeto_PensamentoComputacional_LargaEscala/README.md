# Projeto – Pensamento Computacional para Sistemas de Larga Escala

## Descrição
Este projeto foi desenvolvido como parte da disciplina Pensamento Computacional, com a Profa. Kadidja Valéria. O foco é a concepção de um sistema de larga escala utilizando os pilares do pensamento computacional e engenharia de software.

## Objetivos
* Relacionar engenharia de software e pensamento computacional.
* Reconhecer princípios e padrões relevantes para sistemas de larga escala.
* Identificar dificuldades reais no desenvolvimento de aplicações complexas.

## Sistema Proposto
**Nome do Sistema:** Rede Social Global
**Descrição:** Aplicação para conexão entre usuários, permitindo postagens de mídia e interações em tempo real. Integra:
* Cadastro e autenticação segura.
* Feed dinâmico de conteúdos.
* Sistema de microsserviços para escalabilidade total.

## Pensamento Computacional Aplicado
* **Decomposição:** Divisão em módulos de Autenticação, Feed e Interações.
* **Reconhecimento de Padrões:** Lógica de notificações e fluxo de login padronizados.
* **Abstração:** Foco no fluxo lógico de dados, ignorando detalhes de hardware.

## Metodologia de Desenvolvimento
* **Metodologia:** Scrum (Sprints de 2 semanas).
* **Ferramentas:** GitHub para documentação e versionamento.

## Desafios Identificados
* Escalabilidade para milhões de usuários simultâneos.
* Segurança de dados (Princípios de Saltzer & Schroeder: Menor Privilégio e Bloqueio por Padrão).
